读法：健康牛市中，所有广度指标应与SPY同步创新高。SPY涨但广度不跟 = 背离（Divergence）→ Stage 2警戒。
核心逻辑：S&P 500是市值加权指数，少数巨头（NVDA/AAPL/MSFT）可以独自撑住指数，而其余数百支股票悄悄恶化。广度指标对每支股票等权，能立即暴露这种分裂。
NYSE 累计A/D Line（INDEX:ADDN）= $NYAD
ⓘ
ADDN — 累计涨跌线（等价于$NYAD） 每日把"涨家数−跌家数"累加，形成一条反映市场内部健康度的长期曲线。这是最经典的广度指标，StockCharts上的$NYAD与此完全相同。
信号解读：
📗 ADDN与SPY同步创新高：广度健康，上涨有效
📕 SPY创新高但ADDN低于前次高点：顶背离，少数大盘股撑盘，内部已在恶化
📕 ADDN跌破20日均线：中期广度转弱确认

历史案例：2000年科网泡沫顶部，ADDN在纳指见顶前约18个月开始背离；2007年金融危机前约6个月同样背离。这是最有历史验证的长期顶部信号。
NYSE 每日涨跌差值（USI:ADVDEC.NY）
ⓘ
ADVDEC.NY — 每日净涨跌家数 ADDN的原材料：每天NYSE涨家数减去跌家数的原始差值。正数=当日涨多跌少，负数=跌多涨少。

ADDN vs ADVDEC.NY的关系：
ADVDEC.NY = 每日原始值（柱状图）
ADDN = ADVDEC.NY的历史累计（折线图）

信号解读：
📗 连续多日大幅正值（>+1000）：市场参与度极广，上涨健康
📕 指数涨但ADVDEC.NY持续低迷或负值：涨势依赖少数股票，危险
📕 出现单日极端负值（<-2000）：恐慌性抛售，通常是短期底部信号
% NYSE stocks above 200-day MA（INDEX:MMTH）
ⓘ
MMTH — 长期广度健康度 NYSE全部股票中，收盘高于自身200日均线的比例。这是判断牛熊大方向最重要的广度指标。
信号解读：
📗 >70%：广度健康，牛市参与度广
⚠️ 50–70%：中性，观察方向
📕 <50%：过半股票处于下跌趋势，即使指数还在高位也已内部恶化

配合ADDN使用：ADDN背离 + MMTH跌破50% = 两个维度共振，Stage 3预警。
% NYSE stocks above 50-day MA（INDEX:MMFI）
ⓘ
MMFI — 中期广度 比MMTH敏感，能更早捕捉市场内部变化，通常比MMTH提前2–4周转向。
信号解读：
📗 >70%：中期动能强
⚠️ 40–70%：中性区间
📕 <40%：市场内部明显走弱
📕 <20%：超卖，配合趋势是A/B级加仓时机

关键用法：MMFI从>70%掉头 → 调整启动的常见前兆，比SPY价格早1–2周。
% NYSE stocks above 20-day MA（INDEX:MMTW）
ⓘ
MMTW — 短期广度（最灵敏） 反应最快，适合判断短线超买超卖，常用于判断调整是否结束。
信号解读：
📗 >80%：短线超买，回调概率高
⚠️ 40–80%：正常区间
📕 <20%：短线极度超卖，常是急跌后反弹的起点

三线共振法：MMTW + MMFI + MMTH同时从高位下滑 → 调整信号最可信，三个时间维度同步恶化。
SPY — S&P 500（参照基准）
ⓘ
SPY — 对比基准 将SPY与上方所有广度指标对比，识别背离是本页核心操作。
背离检查三步：
1. SPY是否在创近期新高？
2. ADDN是否同步创新高？
3. MMTH/MMFI是否同步创新高？

三者都创新高 = Stage 1健康
SPY创高但ADDN/MMTH未跟 = Stage 2警戒
SPY创高且ADDN/MMTH明显低于前次高点 = Stage 3预警
% S&P 500 above 200-day MA（INDEX:S5TH）
ⓘ
S5TH — 大盘股广度（聚焦SPX成分） 只统计S&P 500的500支成分股，排除小盘股噪音，更能反映机构持仓标的的健康度。
信号解读：
📗 >70%：大盘股广度健康
📕 <50%：多数成分股走弱，指数被少数巨头（NVDA/AAPL/MSFT）托住

经典案例：2023年上半年S5TH长期低于50%，而SPY因"科技七巨头"撑盘表面正常——这种分裂是结构性风险的典型特征。
IWM / SPY 比值 — 小盘相对强弱
ⓘ
IWM/SPY — 最早的领先指标 Russell 2000对S&P 500的比值，小盘股对经济周期和流动性最敏感，往往最早反映市场内部变化。
信号解读：
📗 比值上升：小盘跑赢，风险偏好强，参与度广
📕 比值持续下跌：资金向大盘龙头集中，广度收窄

领先性：IWM通常比SPY提前2–4周转弱。比值连续下跌超3周且ADDN同步走弱 → Stage 2入场信号最可靠的组合之一。
SPY — S&P 500
ⓘ
SPY — S&P 500大盘 市值加权，受AAPL/MSFT/NVDA等超大盘股主导。

看什么：20日均线方向 + 是否在50日线上方。死叉（50日穿120日向下）= D级信号激活区。
QQQ — Nasdaq 100
ⓘ
QQQ — 科技/成长股代表 集中度更高，科技股权重极大，比SPY更容易出现超涨超跌。

看什么：QQQ与SPY同向则市场一致性强；QQQ先于SPY转弱，是市场由攻转守的信号。
IWM — Russell 2000
ⓘ
IWM — 小盘股晴雨表 Russell 2000包含2000只小盘股，对经济周期和利率更敏感，是市场风险偏好的真实反映。

看什么：IWM率先破位（跌破20日线）而SPY还在高位 → 2–4周后SPY跟上的概率很高。这是你最值得盯的领先指标。
板块轮动信号：当XLU/XLP/XLV（防御）开始跑赢XLK/XLY（进攻），说明资金开始撤出风险资产 → Stage 2–3预警。

XLK / XLU 比值 — 攻防比
ⓘ
XLK/XLU — 进攻vs防御 科技ETF除以公用事业ETF的比值，直观显示市场当前处于进攻还是防御模式。

信号解读：
📗 比值上升/持平：风险偏好稳定，无需警惕
📕 比值掉头向下：资金从科技流向公用事业 = 防御轮动确认
📕 若同期SPY还在高位 → 典型的顶部特征，历史上多次出现于大调整前2–6周
XLK — Technology
ⓘ
XLK — 科技板块（进攻型）
权重最高成分：AAPL、MSFT、NVDA。牛市领涨，顶部时往往最后见顶，但下跌时也跌最猛。
XLU — Utilities
ⓘ
XLU — 公用事业（防御型）
稳定高股息，对利率敏感。当资金从科技流入XLU时，是市场风险偏好降低的信号。XLU连续跑赢SPY超过2周 → 防御轮动确认。
XLP — Consumer Staples
ⓘ
XLP — 必需消费品（防御型）
食品、饮料、日用品，无论经济好坏都有需求。跑赢SPY = 防御轮动信号。通常与XLU同向，两者都强 = 防御信号更可靠。
Risk-off核查清单：VIX飙升 · HYG下跌 · DXY急涨 · GLD/SPY比值上升 · TLT上涨 → 多个同时出现即为风险信号共振。
VIX — 恐慌指数
ⓘ
VIX — CBOE波动率指数 衡量市场对未来30天SPX波动的预期，俗称"恐慌指数"。

信号解读：
📗 VIX <15：市场极度平静，甚至自满（反向风险！）
⚠️ 15–20：正常波动
📕 20–30：市场开始紧张
📕 >30：恐慌模式，通常对应急跌段

特别注意：VIX从极低位（<13）快速上升超过3点，往往是调整开始的先行信号，比SPY价格更早。
HYG — 高收益债ETF
ⓘ
HYG — 信用市场健康度 追踪垃圾债价格。机构在风险暴露前往往先减持信用风险，HYG是"聪明钱"的踪迹。

信号解读：
📗 HYG与SPY同涨：信用市场健康，股市上涨有支撑
📕 HYG下跌而SPY还在涨：信用利差扩大，机构开始避险

历史规律：2007年、2020年COVID前，HYG都提前SPY几周开始走弱。这是最可靠的宏观风险信号之一。
USDX — 美元指数
ⓘ
USDX — 美元走势 美元对一篮子主要货币的综合汇率指数。

信号解读：
📕 DXY急速上涨：美元流动性收紧，风险资产承压，新兴市场首当其冲
📗 DXY温和下跌：流动性宽松，利好风险资产

注意：并非DXY涨就是坏事，需看上涨速度。短期内快速上涨3%+ = 流动性危机信号，历史上常伴随股市大跌。
GLD / SPY — 黄金相对强弱
ⓘ
GLD/SPY — 避险情绪指标 黄金ETF除以SPY的比值，显示资金在黄金和股票之间的相对偏好。

信号解读：
📗 比值下跌：股票跑赢黄金，风险偏好强
📕 比值上升：黄金开始跑赢股票，避险情绪升温
📕 比值快速拉升 + VIX同步上升 = 避险共振，调整大概率已经开始
TLT — 20年期美债ETF
ⓘ
TLT — 长债避险指标 追踪20年以上美国国债。当股市恐慌时，资金涌入国债，TLT上涨。

信号解读：
📕 TLT上涨 + SPY下跌 = 典型flight-to-safety，调整确认
📗 TLT下跌 + SPY上涨 = 风险偏好强，股市上涨有效

复杂情况：当通胀预期高时，TLT可能不跟涨（债券熊市）。此时需要改看VIX和HYG来判断风险情绪。
